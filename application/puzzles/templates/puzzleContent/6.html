Dynamic Programming Weights


You are given a graph with n (n <= 45) and m (m <= 80) edges, and a positive integer k (k <= 5 ). For any path in the graph, define the value of the path as the k-th largest weight out of the edges in the path (if the path has <k edges, the value is 0). Find the minimum value of a path from node 0 to node n-1.

Input:

The first line contains two integers, n and k.

The second line contains integer m.

The next m lines contain 3 integers u, v, w each, meaning node u is connected to node v by an edge with weight w (w <= 50000).

Example:

Input:

6 3

6

0 1 10

1 2 3

2 5 8

0 3 5

3 4 6

4 5 4

Output:

3

Explanation:

There are two paths from 0 to 5, 0-1-2-5 and 0-3-4-5. For the first path, the edge weights are 10, 3, 8 so the 3rd largest weight is 3. For the second path, the edge weights are 5,6,4 so the 3rd largest weight is 4. The first path has a lower value of 3 so that is the answer.

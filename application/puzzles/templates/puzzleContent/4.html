K-th Correct Bracket Sequence

Recall that a correct bracket sequence is one where you can put numbers and signs in between, to get a valid arithmetic expression.
For example (()) and ()(()) are valid and (()))() is not. Assume that ‘(‘ is lexicographically smaller than ‘)’ and compare
correct correbracket sequences as regular strings. Your task is by a given N and K, to find the K-th lexicographically smallest 
correct bracket sequence that contains N opening brackets and N closing ones.

Constraints:

1 <= N <= 5000

1 <= K <= 1018

Input:

On the first line of the input are the numbers N and K (in that order)

Output:

On the first line of the output you should output the
K-th correct bracket sequence of length 2N

Examples:

Input:                 Output:

3 3                       (())()


Input:                 Output:

10 723                 ((((()(())(())))))()
